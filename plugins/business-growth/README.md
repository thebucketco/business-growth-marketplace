# business-growth

Six expert business-growth skills built from 20 curated, battle-tested prompts, consolidated and organized along the growth lifecycle: Understand → Package → Launch → Attract → Keep → Grow.

| Skill | Playbooks |
|---|---|
| market-intelligence | Market opportunity (TAM/SAM/SOM, bottom-up), customer persona deep dive, competitor breakdown with differentiation gaps |
| offer-positioning | Brand positioning, value proposition optimization, offer design, 3-tier pricing, revenue stream expansion |
| go-to-market | Phased GTM launch, lead generation engine with CAC lens, funnel architecture & leak audit |
| content-messaging | 30-day content strategy (40/40/20) with hooks deep-dive, email marketing system |
| conversion-retention | CRO with the MECLABS heuristic, onboarding/retention/LTV around Time-to-Value |
| scale-systems | ICE-scored growth experiments, automation & AI integration, scaling roadmap (Theory of Constraints) |

Shared behaviors across all skills: business context collected once and reused; four-leak diagnostic (awareness / conversion / retention / monetization) presented as an interactive question when the request is vague; web search for real market data with auditable math; quality-over-quantity caps; every response ends with "Your next step".
