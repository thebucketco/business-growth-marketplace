# Ju's Business Growth Marketplace

A Claude Code / Cowork plugin marketplace hosting the **business-growth** plugin.

## Install

```
/plugin marketplace add <your-github-username>/business-growth-marketplace
/plugin install business-growth@ju-business-growth
```

## What's inside

The `business-growth` plugin ships six skills covering the full growth lifecycle (Understand → Package → Launch → Attract → Keep → Grow):

| Skill | Playbooks |
|---|---|
| market-intelligence | Market opportunity (TAM/SAM/SOM), customer persona deep dive, competitor breakdown |
| offer-positioning | Brand positioning, value proposition, offer design, tiered pricing, revenue streams |
| go-to-market | Phased GTM launch, lead generation with CAC lens, funnel architecture & leak audit |
| content-messaging | 30-day content strategy, viral hooks, email marketing system |
| conversion-retention | CRO (MECLABS heuristic), onboarding/retention/LTV around Time-to-Value |
| scale-systems | ICE-scored growth experiments, automation, scaling roadmap (Theory of Constraints) |

See `plugins/business-growth/README.md` for details.
